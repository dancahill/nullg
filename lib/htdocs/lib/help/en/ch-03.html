3 - Bookmarks

3.1 - The Bookmarks List

          The Bookmarks list is a shared listing of favourite and/or more commonly used web site URLs (Universal Relocation Link). It can also be used for ftp and telnet links if your browser supports the option.

          In the bookmark list view, you will be presented with a sorted list of all URLs which have been added to the database. In the left column of each bookmark, an 'edit' link will be available if you wish to edit a bookmark. The right column is the bookmark itself. Clicking on the web site link will open that bookmark in a new window.

3.2 - Adding and Editing Bookmarks

          To add a new bookmark, click on 'NEW BOOKMARK' at the top of any page in the bookmarks section. To edit a bookmark, click on 'edit' beside a bookmark in the bookmarks list view.

    Bookmark Folder

          The 'Bookmark Folder' option allows you to choose the folder in which to store the bookmark.

    Bookmark Name

          The 'Bookmark Name' should be the name or a very short description of the site to which the bookmark is pointing to.

    Bookmark Address

          The 'Bookmark Address' should be the fully qualified URL of the site. Addresses in the format of 'http://www.nowhere.com/' and 'ftp://ftp.nowhere.com/' are valid. Note that omitting the 'http://' part of the address and simply entering 'www.nowhere.com' is likely to create a non-functional bookmark.
