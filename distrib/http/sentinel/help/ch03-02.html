Chapter 3. The Calendar

Viewing and Editing Events

      The event viewer and the event editor can both be accessed by several means, two of which are the calendar Day View and Month View. Both the viewer and the editor display the same information, however the editor can be used to add, edit and delete events.

Event Name

      The 'Event Name' is a brief one line descriptive name which should be used to distinguish this event from others.

Assigned By

      The 'Assigned By' field is used to indicate the name of the user who assigned this event (or reassigned it). This field is automatic.

Assign(ed) To

      The 'Assign(ed) To' field indicates who the current owner of the event is. If this event is marked private, then only this user will be able to view the event. In edit mode, if this field is left blank, then the event will be assigned to the current user.

Date

      The 'Date' field indicates the day on which the event is to occur.

Time

      The 'Time' field indicated the time at which the event is to start.

Duration

      The 'Duration' field indicates the approximate length of time (in minutes) which the event is expected to span.

Status

      The 'Status' field indicates the current status of the event. This can be set to anything, however marking events with a status of 'In Progress' or 'Completed' may be more helpful.

Privacy

      The 'Privacy' field indicates whether this event is publicly viewable by all users, or if access is restricted to the user to whom the event has been assigned.

Priority

      The 'Priority' field indicates what importance has been placed on this event.

Reminder

      The 'Reminder' field indicates how long before an event Sentinel Groupware will start reminding the event owner.

Repeat Event

      The 'Repeat Event' field indicates how often this event will repeat starting from the event's start date.

Repeat Until

      The 'Repeat Until' field is used to control the end date of repeating events.

Notes

      The 'Notes' field is generally used to record any notes or reminders for an event which do not fit into any of the other fields.
