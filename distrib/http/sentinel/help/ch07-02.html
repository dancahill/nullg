Chapter 7. Links

Adding and Editing Links

      To add a new link, click on 'ADD HYPERLINK' at the top of any page in the links section. To edit a link, click on 'edit' beside a link in the links list view.

Link Name

      The 'Link Name' should be the name or a very short description of the site to which the link is pointing to.

Link Type

      The 'Link Type' should be a short description of the category of this link.

Link Address

      The 'Link Address' should be the fully qualified URL of the site. Addresses in the format of 'http://www.nowhere.com/' and 'ftp://ftp.nowhere.com' are valid. Note that omitting the 'http://' part of the address and simply entering 'www.nowhere.com' is likely to create a non-functional link.
