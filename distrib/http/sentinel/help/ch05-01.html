Sentinel Groupware
Chapter 5. Files

Managing the File Section

      The files section is a simple system which can be used as a small repository for files frequently accessed from remote locations. If your network is configured for network file system access, you may not need to use the Sentinel Groupware file system.

      The files section does not offer tools for directory management or for moving or deleting files. These functions are best done with the use of standard tools such as Windows Explorer, or from the command line.

      If you add subdirectories in the file directory, Sentinel Groupware will allow full usage of these directories. This may be a very useful way of keeping files sorted if your file base starts to get large or difficult to manage.
