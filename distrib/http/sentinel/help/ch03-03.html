Chapter 3. The Calendar

Viewing and Editing Tasks

      The task editor can be accessed by the calendar Day View.

Task Name

      The 'Task Name' is a brief one line descriptive name which should be used to distinguish this task from others.

Assigned By

      The 'Assigned By' field is used to indicate the name of the user who assigned this task (or reassigned it). This field is automatic.

Assign(ed) To

      The 'Assign(ed) To' field indicates who the current owner of the task is. If this task is marked private, then only this user will be able to view the task. If this field is left blank, then the task will be assigned to the current user.

Privacy

      The 'Privacy' field indicates whether this task is publicly viewable by all users, or if access is restricted to the user to whom the task has been assigned.

Priority

      The 'Priority' field indicates what importance has been placed on this task.

Due Date

      The 'Due Date' field indicates the day by which the task is expected to be completed.

Status

      The 'Status' field indicates the current status of the task. This can be set to anything, however marking tasks with a status of 'In Progress' or 'Completed' may be more helpful.

Task Notes

      The 'Task Notes' field is generally used to record any notes or reminders for a task which do not fit into any of the other fields.
