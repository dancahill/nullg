Chapter 7. Links

The Links List

      The links list is a shared list of hyperlinks which can be used as a shared listing of favourite and/or more commonly used web site URLs (Universal Relocation Link). It can also be used for ftp and telnet links if your browser supports the option.

      In the link list view, you will be presented with a sorted list of all URLs which have been added to the database. In the left column of each link, an 'edit' link will be available if you wish to edit a link. The center column is the actual link itself. Clicking on the title of the web site link in the center column will open that link in a new window. In the right column is a brief one line description of the type of link.

      At the bottom of the links list page is a simple form which you can use to do simple searches on one of a number of common search engines. Clicking on 'Search' will open the query results in a new window.
