Sentinel Groupware
Chapter 11. Web E-mail

Configuring E-mail

      If you have not already done so, you will need to configure your e-mail settings before using the e-mail system. To view or edit your e-mail settings, click on 'profile' on the top menu, and then 'edit profile'. On this form, you will be presented with options as follows:

Password

      This is your system password used for logging onto Sentinel Groupware. You can change your login password at any time from here.

E-mail Address

      This is the full return address which should be attached to your outgoing e-mail in the form of username@host.com.

E-mail Username

      This is the login name which you use to connect to your pop3 server.

E-mail Password

      This is the password which you use to connect to your pop3 server.

POP3 Server

      This is the host name of your pop3 server. This is usually a host name which looks something like pop3.host.com.

SMTP Server

      This is the host name of your SMTP server. This is usually a host name which looks something like smtp.host.com.

Using E-mail

      Once your e-mail settings are configured, getting started with the web e-mail interface should be as easy as clicking on the 'Web E-mail' link on the main menu.
