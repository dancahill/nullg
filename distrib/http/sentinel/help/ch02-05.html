Sentinel Groupware
Chapter 2. Administration

The User Editor

      The user editor can be used to add, modify, or delete user accounts. All aspects of a Sentinel Groupware user's profile, including login information, security access levels, e-mail settings, and personal information can be changed from the user editor.

Username

      This is the login name which is assigned to the user. Each user must have a unique username.

Password

      This is the user's login password. Each user should be advised to choose a password which is both easy to remember, and hard to guess. Passwords are encrypted before being stored in the Sentinel Groupware database. If a user forgets their password, then the administrator will have to assign a new password to the user.

E-mail Address

      This is the e-mail address which will be used in the From: field on all outgoing Web E-mail.

E-mail Username

      This is the username to use when logging on to the POP3 e-mail server.

E-mail Password

      This is the password to use when logging on to the POP3 e-mail server. Note that this password is stored in the database in cleartext. If you have any concerns about the privacy of your e-mail, it is suggested that you quit the web e-mail system when not using it (thus removing the e-mail password from the database).

POP3 Server

      This is the fully qualified domain name of the POP3 server to use when retrieving e-mail.

SMTP Server

      This is the fully qualified domain name of the SMTP server to use when sending e-mail.

Administration

      All administrative functions in Sentinel Groupware require level 4 security. Setting this to anything less than level 4 will disallow the user from accessing any of the administrative functions. Setting this to level 0 will remove the Administration menu option entirely.

Calendar

      This setting controls the access level granted to the user for the Calendar section.

Level 0 will remove the Calendar menu option entirely.
Level 1 or higher grants a user the ability to view all public tasks and events as well as all private tasks and events assigned to the user.
Level 2 or higher grants a user the ability to assign new tasks and events.
Level 3 or higher grants a user the ability to modify and delete existing tasks and events.

Contacts

      This setting controls the access level granted to the user for the Contacts section.

Level 0 will remove the Contacts menu option entirely.
Level 1 or higher grants a user the ability to view all contacts and companies.
Level 2 or higher grants a user the ability to add new contacts and companies.
Level 3 or higher grants a user the ability to modify and delete existing contacts and companies.

Files

      This setting controls the access level granted to the user for the Files section.

Level 0 will remove the Files menu option entirely.
Level 1 or higher grants a user the ability to download files.
Level 2 or higher grants a user the ability to upload new files.

Forums

      This setting controls the access level granted to the user for the Forums section.

Level 0 will remove the Forums menu option entirely.
Level 1 or higher grants a user the ability to view all forums and replies.
Level 2 or higher grants a user the ability to reply in forums.
Level 3 or higher grants a user the ability to create new forums.

Links

      This setting controls the access level granted to the user for the Links section.

Level 0 will remove the Links menu option entirely.
Level 1 or higher grants a user the ability to view all links.
Level 2 or higher grants a user the ability to add new links.
Level 3 or higher grants a user the ability to modify and delete existing links.

Messages

      This setting controls the access level granted to the user for the Messages section.

Level 0 will remove the Messages menu option entirely.
Level 1 or higher grants a user the ability to read and delete personal messages.
Level 2 or higher grants a user the ability to send messages.

Orders

      This setting controls the access level granted to the user for the Orders section.

Level 0 will remove the Orders menu option entirely.
Level 1 or higher grants a user the ability to view all orders and products.
Level 2 or higher grants a user the ability to add and modify orders.
Level 3 or higher grants a user the ability to delete orders, and add, modify and delete products.

Queries

      This setting controls the access level granted to the user for the Queries section.

Level 0 will remove the Queries menu option entirely.
Level 1 or higher grants a user the ability to run predefined queries.

      Since running arbitrary SQL queries could present a security risk, level 4 access is required to add or run new queries.

Webmail

      This setting controls the access level granted to the user for the Web E-mail section.

Level 0 will remove the Web E-mail menu option entirely.
Level 1 or higher grants a user full use of the web e-mail system.

Surname

      This is the surname or last name of the user.

Given Name

      This is the given name or names of the user.

Title

      This is the job title of the user.

Division

      This is division the user works in.

Supervisor

      This is name of the user's supervisor.

Address

      This is the street address of the user.

City, Province

      This is the city and province the user lives in.

Country

      This is the country the user lives in.

Postal Code

      This is postal code of the user's address.

Home Phone

      This is the home phone number of the user.

Work Phone

      This is the work phone number (and extension) of the user.

Cell Phone

      This is the cell phone number of the user.

FAX

      This is the FAX number of the user.

Pager

      This is the pager number of the user.

Date of Birth

      This is the user's date of birth.

Hire Date

      This is the date the user was hired.

S.I.N. (Social Insurance Number)

      This is the social insurance number (or social security number) of the user.

Active

      This specifies whether the user is currently active in the company or not.

Notes

      Any notes or memos about the user should go here.
