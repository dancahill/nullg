Chapter 4. Contacts and Companies

Viewing and Editing Companies

      The company viewer can be accessed from the company search/list or from any contact view with an associated company. Both the viewer and the editor display the same information, however the editor can be used to add, edit and delete companies.

      Each company view also has a link below the table which can be easily used to find all contacts that are associated with this company.

Company Name

      The 'Company Name' field is the name by which the company is best known.

Company Type

      The 'Company Type' field is used to indicate the primary function of the company. This can be anything, such as 'Retail Distributor', 'Legal Firm', etc.

Phone Number

      The 'Phone Number' field is the primary phone number (and extension) used to contact this company.

FAX Number

      The 'FAX Number' field is the primary FAX number for this company.

Primary Contact

      The 'Primary Contact' field is used to associate the company with a contact in the contacts table. In view mode, clicking on the contact name will take you directly to a view of that contact.

Address

      The 'Address' field is the street address of the company.

City, Province

      The 'City, Province' field is the city and province (or state) in which the company is located.

Country

      The 'Country ' field is the country in which the company is located.

Postal Code

      The 'Postal Code' field is the postal code (or zip code) of the company's mailing address.

Web Site

      The 'Web Site' field is used to record the main web site URL of the company. In view mode, clicking on the web site address will take you directly to that company's web site.
