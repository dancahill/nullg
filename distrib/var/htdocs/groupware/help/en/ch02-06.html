Chapter 2. Administration

The User Group Editor

      The group editor can be used to add, modify, or delete user groups. Groups can best be defined as logical subcategories of users. The primary use of user groups can be seen in ownership of records such as contacts or events. In this case, a user may choose to grant a higher priveledge to other users in his/her group, and lesser priveledges to users who are not in the same group.

Group Name

      This is the visible group name used to represent this user group.

Message of the Day

      This is the message of the day which will be displayed for all users in this group immediately after a successful login.

Availability

      Clicking on the [Availability] link will take you to a grid which represents one full normal work week, broken up into one hour blocks. Each block with a checkmark indicates an hour which users in this group may mark in their own profile as available time. Any time block which is not selected will be automatically marked 'unavailable' for all users in this group.
