NullLogic Groupware
Chapter 8. Files

Managing the File Section

      The files section is a simple system which can be used as a repository for files frequently accessed from remote locations. If your network is configured for network file system access, you may not need to use the NullLogic Groupware file system.

      The files section does not offer tools for moving or editing files, however it does allow for the creation and deletion of both files and directories. File access, as with other subsystems, is controllable via ownership and permissions placed on files.

      If you add directories or files in the file directory using other means, NullLogic Groupware will automatically map the files in its database, however the files/directories will not be owned by anyone, will have minimal permissions, and will not be accessible to users who do not have administrative access to the 'files' section.
