Chapter 5. Calls

Viewing and Editing Calls

      The call viewer can be accessed from the calls list, and by listing calls associated with a given contact. Both the viewer and the editor display the same information, however the editor can be used to add, edit and delete calls.

Call Name

      The 'Call Name' is a brief one line descriptive name which can be used to distinguish this call from others.

Assigned By

      The 'Assigned By' field is used to indicate the name of the user who assigned (or reassigned) this call. This field is automatic.

Assign(ed) To

      The 'Assign(ed) To' field indicates the user this call is assigned to. In edit mode, if this field is left blank, then the call will be assigned to the current user.

Contact

      The 'Contact' field indicates the person to whom this call is associated.

Call Date

      The 'Call Date' field indicates the day on which the call occurred.

Call Time

      The 'Call Time' field indicated the time at which the call occurred.

Action

      The 'Action' field indicates the nature of the call, and in many cases, the expected action to be taken. Default actions include; 'Came to see you', 'Phoned', 'Please call', 'Returned your call', 'Wants to see you' and 'Will call again'. Adding and editing call actions is possible via direct manipulation of the database.

Status

      The 'Status' field indicates whether the call is open or closed. A status of 'open' typically means that further action is required in response to this call.

Details

      The 'Details' field is used to record any additional information relevant to the call.
