Chapter 10. Messages

Sending and Replying to Messages

      To send a new message, click on 'SEND MESSAGE' at the top of any page in the message section. This will open a new form and allow you to send a new message. To reply to a message, click on the 'Reply' link in any message view.

To

      The 'To' field is the username to which the message will be sent. If this is left blank, then the message will not be sent.

Subject

      The 'Subject' field is a brief description of the message body. If you are sending a really short message, you may want to use the subject line instead of a message body. This is assuming of course that the message recipient will know the context in which the message was sent.

Message Body

      The message body is a short message to the user. If you need to send messages to multiple recipients, or if you need to attach a file, you may want to consider using e-mail instead. As mentioned before, the messenger system was only designed to be used with short person-to-person text messages.
