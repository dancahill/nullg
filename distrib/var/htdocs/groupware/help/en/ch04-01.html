Chapter 4. The Calendar

Calendar Views

      The calendar has four views in which it can be viewed. These are the "Day View", "Week View", "Month View" and the "Year View". Each of these views is filtered by user. Each calendar view is also filtered by status. The status filter allows the user to view all events/tasks for the selected user, only events/tasks which are open, or only events/tasks which are closed.

Day View

      The first mode, the "Day View", displays a listing of the selected day's events which the current user has permission to view. Each event and task displayed here can be viewed simply by clicking on the record's title.

      To add a new event, click on the time field beside an empty slot. This will bring up a form for adding a new event. To view an existing event, click on the time field beside that event's name. For more information on adding, editing and viewing events, see Viewing and Editing Events.

      The Day View also has a mini calendar view which can be used to easily choose which day you want to view. To change the current day, simply click on the day you want to view. If you want to change the month and/or the year of the mini calendar, you can do so by selecting the desired year and month with the form directly above the mini calendar.

      One other feature of the Day View is that it offers access to the Task List function. Tasks are quite similar to events, however tasks are generally not date-dependant. One notable difference between events and tasks is that tasks do not need to be assigned to a particular user. Unassigned tasks will appear in the task list with an asterisk beside them. If permissions allow, users can reassign tasks to themselves. To add a new task, click on (new). To view, edit or delete a task, click on the task name. For more information on adding, editing and viewing tasks, see Viewing and Editing Tasks.

Week View

      The "Week View" display is similar to the day view, however, instead of displaying events for a single day, this view displays a compressed list of events for an entire week.

Month View

      The "Month View", displays a listing of the selected month's events which the current user has permission to view. Each event and task displayed here can be viewed simply by clicking on the record's title.

      Within each cell of the calendar, a listing of the time and event name will be shown for each viewable event for that day. To view an event, click on the event name, and you will be taken to the view form for that event.

      At the top of the Month View calendar, there is a form which will let you select the month and year of the month being viewed. To change to Day View for any of the dates displayed in the current Month View, simply click on the date, and you will be taked directly to that day's Day View.

Year View

      The "Year View", displays a mini-calendar view for each month in the selected year. Clicking on a month name will take you to a month view of the same month. Clicking on a specific day will take you to a day view of the selected day.
