The Error Log Viewer

      The error log viewer can be used to view the contents of the error.log file which contains a record of errors caused by the SQL server or the NullLogic Groupware Server. Some examples and explanations are given below.

      In this example, the client web browser requested a URL which does not translate into an internal command, and it does not represent a file which can be downloaded by the client. 

	Apr 01 12:05:57 - 127.0.0.1 - Incorrect function call '/groupware/images/logo.gif' by administrator

      This, and similar errors usually indicate a problem with the SQL server itself. In this case, the MySQL server which NullLogic Groupware is trying to connect to is not running. The same error could also be caused if the SQL connection information in the configuration is wrong. Other similar errors may be caused by incorrect SQL query syntax. 

	Apr 01 12:07:46 - MYSQL error: Can't connect to MySQL server on 'localhost' (10061)

      This particular error may be quite common during initial installation if ODBC is chosen as the SQL type, and you have not installed the necessary updates on your system. For more information, read Troubleshooting. 

	Apr 01 12:09:16 - ODBC Error: [IM002][Microsoft][ODBC Driver Manager] Data source name not found and no default driver specified (0)
