Chapter 2. Administration

The Zone Editor

      The zone editor can be used to add, modify, or delete geographical zones. Zoning is an optional feature which allows all users and contacts to be organized into logical subcategories by their geographical location. The primary use of zoning is in the autoassignment feature of the calendar system, where it would be beneficial to automatically assign an event to a user who works in the same region that the contact is located in.

Zone Name

      This is the visible zone name used to represent this region. When assigning a name to a zone, it is helpful to use a short, but descriptive name of the zone.
